## Personal Soundboard Player v0.8.1

### Layout areas — visual editor (Phase C)

- **16:9 live preview** on the Layout areas page — see slot placement for 16:9 and 9:16 clips as you edit.
- **3×3 anchor picker**, margin sliders with **Uniform** toggle, and max size sliders.
- **Mapping preview** — display defaults section shows landscape and portrait ghost slots on the stage.
- **Reorder areas** with ↑ / ↓ in the areas list.

### Dashboard — Play in…

- Video clip menu (⋮) adds **Play in…** — expand to play immediately in any registered layout area.
- **Main ▶ button** always uses the **orientation default** (landscape / portrait mapping from Layout areas settings).
- **Play in…** does not change the default; it only plays once in the chosen area.
- Removed the per-card **Layout area** dropdown (replaced by the menu).

### Fixes

- Clip menu no longer clipped by the card boundary.
- **Uniform** margins checkbox can be toggled off in the layout area editor.

### Upgrade from v0.8.0

Install over v0.8.0. No OBS URL changes. Refresh the dashboard after updating if it was already open.

### Still planned (Layout Stage)

- Per-clip default layout area in the clip editor (Phase D).
- Further polish from [overlay-layout-stage.md](https://github.com/mmartinewski/personal-remote-soundboard/blob/main/docs/overlay-layout-stage.md).
