## Stream Media Board v0.34.4

### Installer

- Relaunch the app automatically after a successful silent install (in-app update or `/VERYSILENT`).
- Retry single-instance startup briefly so the post-install relaunch is not dropped while the old process exits.

### Upgrade from v0.34.3

Use **Check for Updates** in the tray menu, or install this build manually.
