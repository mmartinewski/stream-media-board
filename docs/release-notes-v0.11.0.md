## Stream Media Board v0.11.0

### Dashboard and shell layout

- **Full-width UI**: header, search toolbar, and main content use the full horizontal space of the window.
- **Navigation menu**: vertical ⋮ button opens a right-side overlay with Dashboard, Layout areas, and New clip.
- **Grid toggle** moved next to Stop all on the dashboard toolbar.
- **Settings panel** (global volume, OBS hint) starts collapsed by default on each visit.

### Upgrade from v0.10.0

Install over v0.10.0. No database or overlay URL changes. Refresh the app after updating if it was already open.
