## Stream Media Board v0.34.6

### Edição de clipes de vídeo

- **Thumbnail do frame atual** — na edição de clipes de vídeo, use **Use current video frame** para capturar o quadro exibido no trimmer e ajustar zoom/pan como em qualquer outra imagem.
- **Caret de frame na timeline** — losango âmbar com handle arrastável abaixo da barra; posicione o frame atual dentro do recorte (start/end). Clique na região azul para pular o caret.
- **Preview a partir do caret** — o preview do segmento inicia na posição do caret e volta a loopar a partir dele até o fim do recorte.
- **Preview maior** — altura máxima do vídeo no editor aumentada para facilitar a escolha do frame.

### Upgrade from v0.34.5

Use **Check for Updates** in the tray menu, or install this build manually.
