## Stream Media Board v0.22.0

### Category browse search scope

- **Limit search to this category** checkbox on category and favorites clip views (on by default).
- Uncheck to search **all clips** while staying on the category screen — quick global lookup without leaving browse navigation.
- Placeholder updates to **Search all clips** when the limit is off; preference is kept in the URL (`inCategory=0`).

### Upgrade from v0.21.0

Install over v0.21.0. No database migration required.
